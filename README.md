# Element Inspector  
Простой и красивый инструмент для анализа элементов на странице

## Что это такое

Лёгкое расширение для Chrome, которое позволяет:

- наводить курсор на любой элемент страницы  
- видеть размеры, шрифты, цвета, закругления и другие важные CSS-свойства  
- фиксировать выбранный элемент (клик → тултип остаётся)  
- красивый тёмный тултип с анимациями и стрелкой-указателем  
- подсветка элемента пунктирной рамкой с пульсацией  
- кнопка закрытия у зафиксированного тултипа

Идеально подходит фронтенд-разработчикам, верстальщикам и дизайнерам, которым надо быстро посмотреть стили без открытия DevTools.

## Особенности

- Работает на любых сайтах (включая iframe)  
- Тёмная тема с градиентами и приятными анимациями  
- Автоматически подстраивает положение тултипа, чтобы не вылезал за экран  
- Минималистичный и ненавязчивый интерфейс  
- Переключение одной иконкой в панели расширений  
- Смена иконки (активный/неактивный режим)

## Установка

1. Скачай архив
2. Открой Chrome → chrome://extensions/
3. Включи «Режим разработчика» (вверху справа)
4. Нажми «Загрузить распакованное расширение»
5. Выбери папку с проектом

Готово ✓

## Клавиши и управление

Клик по иконке расширения → вкл/выкл инспектор
Наведение мыши → показ тултипа и подсветки
Клик по элементу → фиксация тултипа
Клик на крестик (✖) → закрытие зафиксированного тултипа

## Текущие показываемые свойства

Размер элемента (ширина × высота)
border-radius
font-family
font-size
font-weight
color (с цветным квадратиком)
background-color (с цветным квадратиком)
